 OLS 회귀 결과를 해석하고
확인하는 방법

1. 모델 개요
종속 변수(Dep. Variable):

여기서는 "quality" (와인 품질)이 종속 변수입니다.

모델:
OLS(Ordinary Least Squares)를 사용하여
회귀 계수를 추정합니다.

관측치(No. Observations):
총 6,497개의 관측치가 사용되었습니다.

2. 적합도 및 모델 요약 지표

모델이 종속 변수인 quality의 분산 중
약 29.2%를 설명한다는 의미입니다.
예시)
와인의 품질(quality)에
영향을 주는 여러 요인을 모델에 포함시켰을 때,
이 모델로 예측한 품질이
실제 관측된 품질의 차이(변동)
중 29.2%만큼 설명되고,

나머지 70.8%는 모델에 포함되지 않은
다른 요인이나
무작위 변동에 의한 것이라는 의미입니다.




Adj. R-squared (조정된 결정계수): 0.291

변수의 수를 고려해 보정한 결정계수로,
거의 동일한 값을 보여
모델에 포함된 설명 변수가
과도하지 않음을 시사합니다.

F-statistic: 243.3 (p < 0.001)
: 모델 전체가 통계적으로 얼마나 의미가 있냐 검사하고,
검사하는 도구, 두 부산을 비교를 한다.
F-statistic = (모델에 의해 설명되는 분산) / 모델로 설명 되지 않는 분산

전체 모델이 유의미함을 나타냅니다.
귀무가설(모든 회귀 계수가 0이다)을
기각할 수 있음을 의미하며,
적어도 한 개 이상의
독립 변수가 종속 변수와 유의미한 관계가
있음을 보여줍니다.

귀무가설 ?(귀무,되돌아오다, 귀, 무: 없다, )
쉽게: 예를들어) 우리가 주장하고 싶은 가설이 틀렸다
또는 아무런 차이나 효과가 없다, 라고 일단 가정하고 시작.
-> 결론, 통계적 검증을 위해, 일부러 내세우는 반대입장이라고 생각하기.

예시)
이 값이 크다는 것은
모델이 단순히 우연에 의한 변동이 아니라,
실제로 종속 변수(여기서는 quality)의
변동을 설명하는 데
충분한 정보를 담고 있다는 의미입니다




AIC/BIC:

모델 선택의 기준으로 사용되며,
여러 모델을 비교할 때 작은 값이
더 좋은 모델임을 나타냅니다.


예시)
AIC (Akaike Information Criterion):
모델의 적합도와 복잡성 사이의 균형을 평가하며,
 값이 낮을수록 더 좋은 모델로 간주됩니다.

BIC (Bayesian Information Criterion):
마찬가지로 모델의 적합도를 평가하지만,
복잡도에 대해 더 엄격한 페널티를 부과하여,
모델 선택시 보다 보수적인 결과를 제공합니다.

이 두 지표 모두 여러 모델을 비교할 때 사용되며,
작은 값이 더 좋은 모델임을 나타냅니다.


coef(Coefficient): 계수,

표준 오차(std err),
(표준 오차는 추정된 계수가 얼마나
불확실한지를 나타내는 지표입니다.
이는 계수의 추정치가 반복 표본 추출 시
얼마나 변동할지를
나타내는 표준편차로, 계수의 신뢰도를 평가하는 데
사용됩니다.
작을수록 계수 추정치의 정밀도가
높다고 볼 수 있습니다.)

t-값,
의미:
t-값은 추정된 계수를 그 표준 오차로 나눈 값입니다.
이 값은 계수가 0이라는 귀무가설을 검정하는 데 사용되며,
절대값이 클수록 해당 계수가 0이 아닐 가능성이 높음을 의미합니다.

p-값 (probability : 확률)
 p-값 (p-value)
의미:
p-값은 귀무가설이 참일 때,
관측된 t-값(또는 그보다 극단적인 값)이
나타날 확률을 나타냅니다.
일반적으로 p-값이 0.05 미만이면,
귀무가설을 기각하고 해당 계수가
통계적으로 유의미하다고 판단합니다.
